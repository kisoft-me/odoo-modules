Frequently Asked Questions

What types of documents can I upload?

You can upload various types of documents, including PDFs, images, spreadsheets, and other commonly used file formats. The system allows you to store any file that might be relevant to a contact, such as contracts, invoices, or personal identification documents.

Is there a file size limit for uploads?

Yes, the file size limit for uploads depends on your Odoo configuration and server settings. Check with your system administrator to confirm the maximum allowed size for file uploads.

What is the purpose of this app?

This app helps you efficiently manage and organize documents related to specific contacts in Odoo. You can upload, track, and retrieve important files associated with customers, vendors, or other contacts.

How do I associate a document with a contact?

To associate a document with a contact, simply navigate to the contacts record, and use the "Upload Document" option in the documents section. You can then attach any relevant files.
